# 🍔 Burger Sales Dashboard | SQL & Power BI

## 📌 Project Overview

The Burger Sales Dashboard is a Business Intelligence project built using SQL and Microsoft Power BI. It analyzes burger sales data to provide insights into sales performance, revenue, product popularity, store performance, and customer trends through interactive visualizations.

The dashboard helps businesses monitor key performance indicators (KPIs), identify top-performing products and stores, and make data-driven decisions.

---

## 🚀 Features

- Interactive Power BI dashboard
- Sales performance analysis
- Revenue analysis
- Top-selling burger analysis
- Store-wise performance
- Location-wise sales analysis
- Monthly sales trend visualization
- Dynamic filtering using slicers
- KPI cards for quick insights
